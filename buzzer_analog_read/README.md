<h4>Descrição:</h4>
Controlar a potência do som do buzzer com um potenciômentro.
<h4>Componentes:</h4>
 - 1x Arduino Uno;
 - 1x Buzzer;
 - 1x Potenciômetro 5Kb;
 - 1x Protoboard;
 - 1x Resistor;
 - 5x Jumpers;
 - <b>Código fonte: </b><a href="">buzzer_analog_read.ino</a>
