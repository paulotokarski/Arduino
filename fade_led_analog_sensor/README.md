#### Definição

Controle a luminosidade do seu Led com um sensor analógico.

#### Componentes

 - 1x Arduino Uno;
 - 1x Led;
 - 1x Resistor;
 - 1x Protoboard;
 - 1x Potenciometro 5 Kb;
 - 5x Jumpers;
